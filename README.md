### 🌌 About Me
I am an AI/ML Engineer and Computer Vision developer dedicated to building local, privacy-first reasoning systems and real-time edge intelligence solutions. My work spans custom RAG architectures, neural search, and advanced object detection pipelines. I love solving complex structural data problems and bringing visual perception to the edge.

- 🧠 **Currently focusing on**: Local, private RAG pipelines, graph representation of text, and low-latency computer vision alerts.
- 🛠️ **Deep dive projects**: Built a local hierarchical tree RAG system from scratch and deployed YOLOv11 for safety detection.
- 🎮 **Fun fact**: Gamer at heart, inspired by stories of stealth, freedom, and complex systems.
